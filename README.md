### Description
A tiny little companion for my mac/desktop,
Multiple profiles using layers in VIAL (not sure of qmk configurator alternative), for different tasks like 3D modelling with easy-to-reach shortcuts for things requiring multiple keys to be pressed, a general layer with screenshot commands (my hands are small so mac screenshot is hard to reach), and other simple automations like keys to open different apps/automations. M

### Requirements
- QMK firmware (UF2)
- QMK configurator/VIAL (I am using VIAL)

|No.|Quantity|Item|
|---|--------|----|
